# 📌 chrome_extension_emoji_app (구글 크롬 확장 프로그램 이모지 앱)  
  
### 📝 앱 소개  
구글에서 제공하는 브라우저 크롬의 확장 프로그램에서 사용할 수 있는 프로그램을 개발했습니다.  
바로! 이모지 앱인데요.  
이 앱은 블로그나 포스팅을 할 때 쓰는 이모지를 익숙하지 않은 키보드 단축키, 번거로운 외부 사이트 이용을 줄이기 위해 만들어졌습니다!  
말하자면 귀찮은 과정을 피하고자 만든 이모지 앱입니다!   
자주 쓰이는 이모지들로 구성을 했습니다.  
추후 다른 이모지도 많이 넣을 생각입니다!  
  
### 🎯 개발 목표  
확장 프로그램이라는 플랫폼에서 배포 경험을 쌓기  
바닐라 javascript 사용 -> react를 주로 사용해서 오는 영양분( javascript 지식 )의 부족을 해결

💡이벤트 위임
